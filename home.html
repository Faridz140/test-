    1.Rancangan Desaign Web Sistem
    Rancangan Web Desaign adalah Aplikasi berbasis browser menjadi lebih umum,dan kelangsungan hidup mereka dalam pengunaan terus meningkat.namun tetap saja mereka datang dengan tunjangan yang biasanya tidak anda temukan dalam perangkat lunak biasa.

    2.Banner
    banner adalah heading pada tampilan website yang berfungsi untuk sebagai papan iklan sebuah website kita supaya website kita bisa di kenal oleh orang lain.

    3.-Menu Top
    Menu Top adalah Element pada html yang di buat untuk mempersentasikan link navigasi.Bisa di bilang wadah link yang akan mendirect ke halaman lain.

    4.Konten
    Konten adalah Informasi yang tersedia melalui media atau produk elektronik.Penyampalan konten dapat di lakukan melalui berbagai mediaseperti internet,televisi,cd audio dan lain sebagainya.

    5.Footer
    Footer adalah bagian dasar atau paling awah pada sebuah website,Fungsi utamanya adalah sebagai kakidan berisi informasi hak cipta kepemilikan.

    6.Tag-Tag HTML
    Berikut adalah tag html beserta fungsinya:
    html : untuk membaca tanda (tag) kode html
    title : sebagai judul web
    head : untuk judul website
    body : untuk membuat isi halaman web
    a herf : untuk membuat tujuan link
    p : sebagai membuat pragraf
    b : untuk menebal tulisan.
